## Dahupt 命名依据

**起因**

基于国~~内~~外软件在研发之初，会启用一套软件(版本、模块、插件)的命名(昵称/代号)规则，诸如

+ Linux 衍生版 ***Debian*** 的所有开发代号均出自Pixar的电影《玩具总动员》；
